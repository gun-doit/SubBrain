> **라이선스 선택**

1. **Commercial 라이선스**
2. **OpenSource 라이선스**
    1. **GPL : 모든 소스코드 공개 ( 비상업용, 기업에서 교육 가능**
    2. **LGPL : 특정 모듈만 사용하지 않으면 소스코드 비공개 가능 ( 상업용 가능 )**

# Qt IDE

1. **Qt Creater ( QC )**
    1. **Qt의 대표적인 IDE, GPL**
    2. **Editor + Qt Quick Designer + Qt Designer 모두 포함**
    3. **디버깅 / SVC / 크로스컴파일 / 가상 에뮬레이터 추가**
2. **Qt Quick Designer**
    1. **Qt QML을 위한 더 나는 UI Designer**
    2. **포토샵과 통합 가능**
3. **Qt Designer**
    
    1. **일반적인 UI Designer, LGP**
    2. **.ui파일을 만드는 용도**
    3. **Editor가 포함되지 않음**
